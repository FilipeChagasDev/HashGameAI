How do it works ?

Whenever you make a move, the algorithm makes a detailed search of each possible end of the game from the current state, calculating the average distance between the current state and the final states in which the agent wins for every move the agent can make after you move. The gap the agent will choose to mark with X will always be the one that gives him the shortest average distance to the states in which he wins. Open the browser console and see the algorithm that describes your activity with logs.

But this algorithm can fail in the mission to always win. You can win the game.
